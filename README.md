# Seka

Marketing site for Seka — AI transformation: strategy, talent, and execution.

Built with Vite, React, React Router, and Tailwind CSS.

## Develop

```bash
npm install
npm run dev      # start the dev server
npm run build    # type-check and build for production
npm run preview  # preview the production build
```

## Pages

- `/` — Home
- `/what-we-do` — Services and engagement models
- `/who-we-are` — Founders and beliefs
- `/work` — Selected work (Clients)
- `/get-a-demo` — Contact
